## How to Capture Screenshots

### Windows (Built-in)
- **Snipping Tool**: Press `Win + Shift + S` to capture a region
- **Full Screen**: Press `PrtScn` to capture the full screen

### Browser Extensions
- **Fireshot** (Chrome/Firefox) - Full page screenshots
- **Awesome Screenshot** - Annotate and capture
- **GoFullPage** - Full page screenshot

## Image Guidelines

- **Format**: PNG preferred (for quality), JPG for smaller files
- **Resolution**: 1920x1080 or higher recommended
- **File Size**: Keep under 2MB per image for faster loading
- **Naming**: Use descriptive lowercase names with hyphens (e.g., `hero-section.png`)
