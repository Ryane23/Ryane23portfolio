# Music Files

This folder contains the music files used by the MusicPlayer component.

## Required Files

1. **mgmt-little-dark-age.mp3** ✅ - Plays on the home page and most routes
2. **nemzzz-track.mp3** ⚠️ - Plays on the /projects page (needs to be added)

## Adding Music Files

To add the Nemzzz track:
1. Obtain the music file (ensure you have proper rights/licenses)
2. Save it as `nemzzz-track.mp3` in this folder
3. The MusicPlayer will automatically pick it up on the /projects route
